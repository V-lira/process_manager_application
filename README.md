# process_manager_application

Process Manager Application
-------------------------
A C# console application that demonstrates process management capabilities including starting external processes, monitoring their execution, and retrieving detailed process information.



Features
---------------------
Process Launch: Start various system applications (Calculator, Notepad, CMD, Paint, Explorer)
Process Monitoring: Real-time monitoring with wait for completion



Detailed Process Info:
----------------------
Process ID (PID)
Process name
Start time
Memory usage
Priority level
Window title
Exit Analysis: Completion time, exit code, and total runtime
Error Handling: Robust exception handling with fallback options
